# 전세계 여행 계획·기록 수요 카탈로그

조사 기준일: 2026-06-09

이 카탈로그는 공식 제품 기능, Marketplace 템플릿, 공개 사용자 불만, 여행
기획 연구를 함께 보고 만든 제품 요구사항 목록이다.

## 제품 우선순위 결론

무료 도구의 품질을 결정하는 최소 단위는 `점 목록`이 아니라
`점 + 선 + 시간 + 기준 계획 + 변경 가능한 현재 계획`이다. 추천, 소셜,
여행 기록보다 먼저 이 실행 모델이 신뢰를 얻어야 한다.

Travelog 유료 전환의 가장 자연스러운 순간은 다음과 같다.

1. 사용자가 장소 주소와 운영시간을 자동으로 채우고 싶을 때
2. 출발 직전 일정 전체를 검증하고 싶을 때
3. 여행 중 지연·운영시간·날씨 변화로 재계획이 필요할 때
4. 여행 후 실제 경로와 사진을 자동으로 기록하고 싶을 때
